# 大学受験速習サイト - 開発ガイド

この日本の高校生向け大学受験学習支援サイトは、React 18 + TypeScript + Tailwind CSS で構築されています。

## プロジェクト概要

**プロジェクト種別**: React + TypeScript モダンWebアプリケーション
**エントリーポイント**: `src/main.tsx` (React アプリケーションエントリー)
**ビルドシステム**: Vite 7.0.0 (高速開発・ビルド)
**スタイリングシステム**: Tailwind CSS 3.4.17 (アトミックCSS)

## アーキテクチャと設計

### ページ構成とルーティング

**ホームページ (`/`)**: 科目選択のランディングページ
- 5つの科目カード（英語、数学、国語、理科、社会）
- 各科目をクリックすると詳細ページに遷移
- サイトの特徴と使い方を紹介

**科目詳細ページ (`/subject/:subjectId`)**: 各科目の学習スケジュール表示
- 3年間×3学期の学習スケジュール表（縦軸：学期、横軸：カテゴリ）
- 大学レベル別参考書推薦システム
- 重要項目の★マーク表示

### 主要機能

#### 1. 学習スケジュール表示システム
- **表のレイアウト**: 学期を縦列、学習項目を横列に配置
- **重要度表示**: 入試頻出項目に★マークで視覚的に区別
- **科目分類**: 
  - 英語：文法・読解・単語
  - 数学：数A・数B・数C・数Ⅰ+A・数Ⅱ+B・数Ⅲ+C（旧課程・新課程両対応）
  - 国語：現代文・古文・漢文
  - 理科：物理・化学・生物
  - 社会：日本史・世界史・地理

#### 2. 大学レベル別参考書推薦システム
- **6段階レベル**: 東大京大、旧帝大、地方国立大、早慶、MARCH、日東駒専
- **完全データセット**: 全科目（理科・社会含む）の参考書データを網羅
- **動的表示**: プルダウン選択で対応する推薦図書が自動表示

### 数学カリキュラムの詳細設計

#### 旧課程対応（数A・数B・数C）の学習スケジュール

**数A の学習スケジュール**
- **1年次**: 場合の数・確率基礎、整数の性質、図形の性質
- **2年次**: 場合の数応用、確率応用、総合演習
- **3年次**: 入試問題演習、過去問演習、最終確認

**数B の学習スケジュール**
- **1年次**: 数列基礎、等差数列・等比数列、シグマ記号・漸化式
- **2年次**: ベクトル基礎、ベクトル内積・成分、平面ベクトルの応用
- **3年次**: 統計的推測基礎、仮説検定、総合演習

**数C の学習スケジュール**
- **1年次**: ベクトル平面・空間、ベクトルの演算、複素数基礎
- **2年次**: 複素数平面、複素数の極形式、複素数応用
- **3年次**: 式と曲線、媒介変数表示、総合演習・入試演習

#### 新課程対応（数Ⅰ+A・数Ⅱ+B・数Ⅲ+C）の学習スケジュール

**数Ⅰ+A の学習スケジュール**
- **1年次**: 数と式・集合と命題、2次関数・2次不等式、図形と計量・三角比
- **2年次**: 場合の数・確率、図形の性質・整数、データの分析
- **3年次**: 総合演習、入試問題演習、過去問・最終確認

**数Ⅱ+B の学習スケジュール**
- **1年次**: 式と証明・複素数、図形と方程式、三角関数
- **2年次**: 指数・対数関数、微分法・積分法、数列・ベクトル
- **3年次**: 統計的推測、総合演習、入試演習・過去問

**数Ⅲ+C の学習スケジュール**
- **1年次**: 基礎準備期間
- **2年次**: 関数・極限、微分法、積分法
- **3年次**: 複素数平面・式と曲線、平面上のベクトル・空間ベクトル、総合・入試演習

## ⚠️ 重要: index.htmlエントリーポイントの保護

**警告**: これはVite + Reactプロジェクトです。`index.html`の以下の行は**絶対に変更しない**でください：

```html
<script type="module" src="/src/main.tsx"></script>
```

**理由**: これがコアエントリーポイントです。変更するとアプリが完全に停止します。

**代わりに**: `src/`ディレクトリで作業してください：
- `src/pages/HomePage.tsx` - ホームページコンポーネント
- `src/pages/SubjectPage.tsx` - 科目詳細ページコンポーネント
- `src/App.tsx` - ルーティング設定
